# JavaScript_encoding
前言：

做项目时发现登录位置参数存在加密，通过调试并编写脚本加密数据的方法比较麻烦，所以编写了一个小工具来批量加密数据。
  
原理：

通过python调用chrome驱动，模拟在chrome控制台执行js的方法。免去了调试的繁琐。
